# Language Cards Web

Language Cards Web — веб-приложение для изучения иностранных языков с помощью карточек, построенное на Nuxt 3.

## Особенности

- Работа с языковыми карточками
- Современный интерфейс
- Интеграция с REST API
- Масштабируемая архитектура

## Стэк

- Nuxt 3
- Vue 3
- TypeScript
- SCSS

# Запуск проекта

Перейдите в [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) для подробного изучения.

## Установка

Убедитесь в том, что установлены зависимости:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Режим разработки

Запустите сервер на `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

## Продакшн

Собрать приложение в режиме "продакшн"

```bash
# npm
npm run build

# pnpm
pnpm build

# yarn
yarn build

# bun
bun run build
```

Локальный просмотр продакшн-сборки

```bash

pnpm preview

yarn preview

bun run preview
```
